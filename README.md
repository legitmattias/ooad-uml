# Object-Oriented Analysis and Design using UML

Coursework from the OOAD course (1DV607) at Linnaeus University. Two team assignments focused on applying design patterns and SOLID/GRASP principles in Java.

This course used **Java** and was centered entirely on object-oriented design rather than web development.

## Assignments

| # | Repository | Description | Key Concepts |
|---|---|---|---|
| A2 | [**Stuff Lending System**](https://github.com/legitmattias/1dv607-ooad-a2) | Console-based lending/contract manager | MVC architecture, GRASP principles, SOLID, encapsulation, UML class/sequence diagrams |
| A3 | [**Blackjack**](https://github.com/legitmattias/1dv607-ooad-a3) | Card game with swappable rule variants | Abstract Factory, Visitor pattern, internationalization (i18n), strategy selection |

## Tech Stack

- **Language**: Java
- **Build**: Gradle
- **Design**: UML (class diagrams, sequence diagrams), design patterns (GoF)
- **Principles**: SOLID, GRASP, DRY
